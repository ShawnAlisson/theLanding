## Getting Started

1. Clone the repo

   ```sh
   git clone https://github.com/ShawnAlisson/theLanding.git
   ```

2. Install NPM packages and Build

   ```sh
   npm i
   ```

3. Run the server in development mode:

   ```sh
   npm run dev
   ```

4. Open [http://localhost:3000] with your browser to see the result.
